## 特性

1. 代码高亮使用的:CodeHighlighter  
  使用其他的需要在pjax:end回调中加入对应的代码高亮渲染函数.
2. 友链插件: Links  
  添加了一个Links友链样式
3. 目录生成: MenuTree(我的修改版)
4. 支持文章头图
5. 自定义首页界面
6. 背景音乐（网易云id或者直接填音乐URL）
7. ...
